# API Endpoint for Report Generation Test

# To use this:
# 1. Ensure your local backend is running on http://localhost:4000.
# 2. Run the REGISTRATION COMMAND below first to create a new user.
# 3. Copy the 'id' from the registration response.
# 4. Run the LOGIN COMMAND to get an access token.
# 5. Copy the 'accessToken' from the login response.
# 6. Replace <YOUR_USER_ID_HERE> and <YOUR_TOKEN_HERE> in the REPORT GENERATION COMMAND.
# 7. Copy and paste the REPORT GENERATION COMMAND into your terminal.

# ---
# REGISTRATION COMMAND: Run this first to create a new user
# ---
curl -X POST http://localhost:4000/api/auth/register \
-H "Content-Type: application/json" \
-d '{
  "email": "test@example.com",
  "password": "test123",
  "name": "Test User"
}'

# ---
# LOGIN COMMAND: Run this after registration to get your access token
# ---
curl -X POST http://localhost:4000/api/auth/login \
-H "Content-Type: application/json" \
-d '{
  "email": "test@example.com",
  "password": "test123"
}'
